### Variáveis de ambiente:

- PORT - A porta da aplicação
- DB_NAME - nome do usuário
- DB_PASS - senha do usuário
- DB_HOST - o host do banco (geralmente localhost)
- DB_PORT - a porta (a padrão é 5432)
- DB_DTBS - nome da base de dados
